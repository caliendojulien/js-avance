Les bases de Javascript

Plusieurs semaines se sont déroulées depuis la dernière fois qu’ils ont fait du Javascript.

On commence par un peu d’histoire et un rappel de l’importance de Javascript de nos jours. On parle très rapidement d’es6 sans rentrer dans les détails pour préparer le module 2. Les concepts restent simples et évidents. Normalement, c’est de l’acquis.

En fin de module, on ajoute npm, webpack et les tests unitaires avec Jest pour que le module ne soit pas trop ennuyant.
